# jogo-sem-nome-in-java-swing
Jogo de luta sem nome com java swing

### Como rodar

Dependencias:
* java (JRE 8)

* git clone
* importar o projeto no seu eclipse
* dar run ou debug na classe src/Controles/Main.java
* seja feliz
